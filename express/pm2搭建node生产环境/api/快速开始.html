申请声明：创建一个配置文件来管理多个应用程序。

  • 文件夹结构，一旦PM2启动，会自动创建这些文件夹。
  • pm2可以启动任何进程类型
    • pm2 start echo.coffee
    • pm2 start echo.py
    • 二进制代码都可以执行
  • 最大内存重新启动：单位可以是K，M，G。
  • 刷新：与restart杀死并重新启动进程相反，reload实现了0秒的重新加载
  • 开发环境和测试环境的切换

更新pm2

  • pm2 save
  • npm install pm2 -g
  • pm2 update
  • 这就是一个全新的pm2系统了
