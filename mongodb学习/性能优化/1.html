mg性能检测

  • mongosniff:从底层监控有哪些命令发送给了mg. sniffer
  • mongostat:可以快速的查看运行中的MG实例的统计信息。每秒更新一次。
  • db.serverStatus:显示更加全面具体的信息，但是一个缺陷是数据是静态的
  • db.stats()查看特定数据库的详细运行状态
  • HTTP 监控接口

mg常用性能方案

  • 创建索引
  • 限制返回结果条数
  • 只查询用到的字段
  • 采取 Capped Collection
  • 采用Server side code Execution命令集
  • 强制使用索引查询
  • 采用优化器
  • 很多优化技巧只有在实际过程中才能体会其精髓，以事实为依据。

影响性能优化的条件

  • 操作系统故障
  • 网络
